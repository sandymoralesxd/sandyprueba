Robótica

 La robótica combina ingeniería mecánica, electrónica y programación para crear máquinas automatizadas que pueden realizar tareas físicas. Un brazo robótico ensamblando piezas en una fábrica de autos.

Inteligencia Artificial

 La inteligencia artificial permite a las máquinas aprender, razonar y tomar decisiones similares a las humanas. Un asistente virtual como Siri o Alexa que entiende y responde a comandos de voz.

Diseño y Desarrollo Web

 El desarrollo web se enfoca en la creación de sitios y aplicaciones accesibles en internet, usando lenguajes como HTML, CSS y JavaScript. Una página web interactiva para una tienda en línea con carrito de compras.
